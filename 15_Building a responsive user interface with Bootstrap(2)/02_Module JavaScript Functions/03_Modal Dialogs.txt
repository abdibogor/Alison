Modal Dialogs

. Configure dialog using classes
	- Container: modal
	. Configure dialog: modal-dialog
	- Content: modal-content
	   .header: modal-header
	   .Content: modal-content
 	   .Footer: modal-footer

.Configure launcher using classes
	-data-toggle="modal"
	-data-target="<id>"

. Configure closer
	- data-dismiss="modal"
